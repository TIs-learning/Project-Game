# MockUp

---
Folder ini berisi rancangan awal (mockup) antarmuka permainan yang digunakan sebagai referensi selama proses pengembangan game.

## Daftar Mockup

| File | Deskripsi |
|------|-----------|
| MockUp Tampilan Game.jpeg | Mockup proses umum pertandingan dan tata letak elemen permainan. |
| MockUp pertandingan game.png | Mockup tampilan utama permainan saat pemain sedang bermain. |

## Tujuan

- Menjadi acuan desain UI/UX (baik perbaikan atau development).
- Membantu tim memahami tata letak setiap halaman.
- Menjadi referensi selama proses implementasi di Unity.

## Catatan Repository

Mockup ini merupakan desain awal dan dapat berubah sesuai hasil evaluasi dan pengembangan proyek.
